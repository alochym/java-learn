# JAVA PACKAGES

- Java packages are just folders
  - Allow you to structure your application to avoid naming conflicts.
  - Grouping of related types.
- [Content of Circle.java](java/src/com/alochym/shape/Circle.java).
- Folder structure

  ```java
  java/src/
  `-- com
      `-- alochym
          `-- shape
              `-- Circle.java
  ```
